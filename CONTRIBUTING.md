# Contributing

Thanks for contributing to the bootcamp lab docs. This repo is documentation-only and must not include solutions or exploit payloads.

## How to contribute

1. Open an issue describing the improvement.
2. Fork the repo and create a feature branch.
3. Submit a pull request with a clear description of changes.

## Style guide

- No spoilers, no payloads, no exploit steps.
- Keep instructions safe, ethical, and educational.
- Prefer clear headings, short paragraphs, and practical steps.
